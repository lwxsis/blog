黑爵-AK35i

黑爵-AK35i

一 灯光控制

  1. FN+Del流光溢彩：整体键盘背光按规则动态流动

    1. 模式一：随波逐流
    2. 模式二：峰回路转
    3. 模式三：涌泉相报
    4. 模式四：波光粼粼

  2. FN+INS水天一色：整体键盘背光颜色同一变化

    1. 模式一：长亮
    2. 模式二：游戏阶梯
    3. 模式三：呼吸
    4. 模式四：暗影前行

  3. FN+PgDN光彩夺目：整体键盘背光为多彩动态变化

    1. 模式一：星光闪烁
    2. 模式二：微波粼粼
    3. 模式三：流光赶月
    4. 模式四：旗彩飞扬

  4. FN+Home一触即发：触发一次键盘，灯光以按键为中心扩散或点亮

    1. 模式一：泛起涟漪
    2. 模式二：踏雪无痕
    3. 模式三：奔逸绝尘
    4. 模式四：六心连锁

  5. FN+PgUP青春时代：整体键盘背光快节奏的有规律的灯光变化

    1. 模式一：龙腾虎跃
    2. 模式二：大鹏展翅
    3. 模式三：数码时代
    4. 模式四：推波助澜

  6. FN+END指点江山：该模式可以自定义每个按键LED的亮灭，默认心性图案

    FN加END进入该模式时默认恢复上一次定义的效果，再次触发进入定义LED模式，指示灯闪烁指示。触发按键可切换LED的亮灭，定义完成后再次触发退出自定义模式，并保存定义结果

  7. FN+↑或FN+↓调节背光亮度，长按持续到最亮或灭

    FN+-或FN++调节流彩速度，长按持续变到最快或最慢

  8. FN+→流动方向向右，FN+←流动方向向左

二 快捷键以及滚轮功能

  1. 独立按键出场默认功能

    A1: 上一曲

    A2: 暂停/播放

    A3: 下一曲

    M: 全键无冲/6键无冲(出厂默认全键无冲相应灯灭，按下切换成6键无冲相应灯亮起)

  2. G键

    G键为功能切换键：默认功能为音量调节模式，相应滚轮向上滚动为音量加，向下滚动为音量减

    按下一次之后为背光亮度调节模式，相应的向上亮度增加，向下亮度减少
