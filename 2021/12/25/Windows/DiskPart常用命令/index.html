DiskPart常用命令

DiskPart常用命令

参考网址

常用命令小记

命令汇总

命令 说明
active 将具有焦点的磁盘分区标记为活动。
add 将选中的简单卷镜像到指定磁盘。
assign 给选中的卷分配一个驱动器号或装入点。
附加 vdisk 将 (VHD) 虚拟硬盘) 装载或 (表面) 以便它作为本地硬盘驱动器显示在主计算机上。
attributes 显示、设置或清除磁盘或卷的属性。
automount 启用或禁用自动装载功能。
break 将选中的镜像卷分为两个简单卷。
clean 从选中的磁盘中删除所有分区或卷格式。
compact vdisk 减少动态扩展虚拟硬盘的物理大小， (VHD) 文件。
convert 将 FAT (FAT) FAT32 卷的文件分配表转换为 NTFS 文件系统，使现有文件和目录保持不变。
create 在磁盘上创建分区、一个或多个磁盘上的卷，或 VHD (虚拟) 。
delete 删除分区或卷。
detach vdisk 停止所选虚拟硬盘 (VHD) 在主计算机上显示为本地硬盘驱动器。
detail 显示有关所选磁盘、分区、卷或虚拟硬盘的信息 (VHD) 。
exit 退出 diskpart 命令解释器。
expand vdisk 将虚拟硬盘 (VHD) 指定的大小。
extend 将具有焦点的卷或分区及其文件系统扩展到磁盘上 (释放) 空间。
filesystems 显示有关具有焦点的卷的当前文件系统的信息，并列出支持格式化卷的文件系统。
format 格式化磁盘以接受Windows文件。
gpt 将 gpt 属性 () 分区，并重点关注基本 GUID 分区表 () 磁盘。
help 显示可用命令的列表或指定命令的详细帮助信息。
import 将一个外磁盘组导入本地计算机的磁盘组。
inactive 将具有焦点的系统分区或启动分区标记为非活动状态，在 MBR 磁盘 (主) 记录。
list 显示磁盘、磁盘中的分区、磁盘中的卷或 VHD (虚拟硬盘) 。
merge vdisk 将差异虚拟硬盘与 VHD (VHD) 其对应的父 VHD 合并。
offline 使联机磁盘或卷处于脱机状态。
online 使脱机磁盘或卷处于联机状态。
recover 刷新磁盘组中所有磁盘的状态，尝试恢复无效磁盘组中磁盘，并重新同步具有过时数据的镜像卷和 RAID-5 卷。
rem 提供一种向脚本添加注释的方法。
remove 从卷中删除驱动器号或装入点。
repair 将故障磁盘区域替换为指定的动态磁盘，修复具有焦点的 RAID-5 卷。
rescan 查找可能已添加到计算机的新磁盘。
retain 准备要用作启动卷或系统卷的现有动态简单卷。
san 显示或设置操作系统 () 存储区域网络策略。
select 将焦点移到磁盘、分区、卷或虚拟硬盘 (VHD) 。
set id 更改具有焦点的分区的 “分区类型” 字段。
shrink 按指定的数量减小所选卷的大小。
uniqueid 显示或设置 GUID 分区表 (GPT) 标识符或主启动记录 (MBR 的主) 签名。
Windows
diskpart windows
