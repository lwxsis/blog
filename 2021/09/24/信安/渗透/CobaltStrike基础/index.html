CobaltStrike基础

重要是熟悉了CobaltStrike基本使用，和在虚拟机中安装服务器，映射到公网IP来获取靶机的办法

参考博客

首先按照博客中的内容进行监听端口的设置，此处是准备的两个端口，一个是需要生成木马对应的端口，一个是监听的本地端口

设置监听端口的具体操作是选择 Cobalt Strike -> Listeners

显示如下界面

点击Add进行设置

参考博客所说的内容

HTTP Hosts：是发送的对应IP地址

HTTP Port(C2)：是发送的对应端口

新建完成后按照 Attacks -> Packages选择对应的监听进行木马的生成

之后在目标机中运行链接成功，整个连接流程是目标机器发送信息到公网IP，公网IP映射到本地端口，通过监听本地端口来进行连接

此处记录help显示的内容
